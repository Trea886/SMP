# 🔮 ESP 能量预言机 Energy Oracle

> 温度 + 光敏 + 红外 + LCD = 一台能"感知你的能量场、预言今日运势"的桌面占卜装置。
> 手一靠近，它就开始读你。

---

## 一、项目概述

### 这是什么

桌上一台小装置，平时呼吸灯待机。你把手靠近（红外+光敏感知），它会"感应你的能量场"——温度波动反映心跳、光线变化反映气场、手势模式反映意念——然后在 LCD 上生成一段个性化预言：**爱情、事业、财运、健康**，四个维度，各有吉凶。

### 为什么做这个

| | |
|:--|:--|
| 🔮 **好玩** | 放桌上朋友来了都想试试，比翻塔罗牌有仪式感 |
| 🧠 **叙事驱动** | 96 条文案库，每次结果不同，重复玩不腻 |
| 🔧 **传感器复用** | 温度/光敏/红外三个传感器，变化率算法直接继承幽灵探测器 |
| 📦 **零新增硬件** | 板载传感器 + LCD 就够，不用额外买模块 |
| ⚡ **一天做完** | 核心逻辑 300 行，三天从写到调完 |

## 二、交互流程

```
┌──────────┐    ┌──────────┐    ┌──────────┐    ┌──────────────┐
│  待机    │ → │  感应    │ → │  摇签    │ → │  显示预言    │
│ 呼吸动画  │    │ 请默念   │    │ 能量分析  │    │ 四维评分卡片  │
└──────────┘    └──────────┘    └──────────┘    └──────────────┘
     ↑                                                │
     └──────────── 10 秒后自动回待机 ←─────────────────┘
```

| 阶段 | 触发条件 | LCD 画面 | 持续 |
|:--|:--|:--|:--|
| **待机** | 上电默认 | 水晶球呼吸动画 + "靠近我" | 无限 |
| **感应** | 红外检测到人 + 光敏波动 | "感知到你的能量场… 请默念问题" | 2 秒 |
| **摇签** | 用户挥手/遮光三次（光敏脉冲计数） | 光线波纹扩散动画 + 进度条 | ~3 秒 |
| **预言** | 摇签完成 | 四维度评分 + 一句话预言 + 幸运数字 | 10 秒 |
| **返回** | 10 秒超时 / 人离开 | 渐变淡出 → 待机 | — |

## 三、核心算法

### 能量种子生成

```python
seed = 0
seed ^= int(temp_delta * 100)       # 温度变化率（你的"心跳"）
seed ^= int(light_delta * 10)       # 光照波动量（你的"气场"）
seed ^= int(wave_interval_ms / 10)  # 挥手节奏（你的"意念频率"）
seed ^= pir_trigger_count           # 红外触发次数
random.seed(seed)
```

> 核心思路：把传感器实时变化量混入随机种子，让每次预言都和当下环境相关——换个人、换个时间，结果会变。

### 四维评分

| 维度 | 英文 | 范围 | 含义 |
|:--|:--|:--|:--|
| 💕 爱情 | Love | 1-10 | 桃花运 |
| 💼 事业 | Career | 1-10 | 贵人/机会 |
| 💰 财运 | Wealth | 1-10 | 偏财/正财 |
| 🍀 健康 | Health | 1-10 | 精力/气色 |

每维 8 条文案（4 吉 + 4 凶），按评分选档位。共 4×8 = 32 条。

### 综合运势文案（彩蛋）

在四维评分之外，额外抽一条一句话预言，如：

- "今日宜社交，穿红色会带来好运。"
- "不适合做重大决定，但适合整理房间。"
- "会收到一条让你微笑的消息。"
- "注意脚下，可能有意外发现。"

共 20 条，随机抽取，不和评分重复。

| 模块 | 型号 | 数量 | 用途 |

## 八、状态机

```
                          ┌──────────┐
                    ┌────→│  IDLE    │←──────────────┐
                    │     │  待机    │                │
                    │     └────┬─────┘                │
                    │          │ PIR触发 + 光敏波动    │
                    │          ▼                      │
                    │     ┌──────────┐                │
                    │     │  SENSE   │                │
                    │     │  感应    │                │
                    │     └────┬─────┘                │
                    │          │ 2秒后                │
                    │          ▼                      │
                    │     ┌──────────┐                │
                    │     │  SHAKE   │  挥手3次/超时   │
                    │     │  摇签    │───→ 回IDLE     │
                    │     └────┬─────┘                │
                    │          │ 摇签完成              │
                    │          ▼                      │
                    │     ┌──────────┐                │
                    │     │  REVEAL  │                │
                    │     │  预言    │                │
                    │     └────┬─────┘                │
                    │          │ 10秒超时 / 人离开     │
                    └──────────┘                      │
```

```python
class Oracle:
    STATE_IDLE   = 0   # 待机：呼吸动画
    STATE_SENSE  = 1   # 感应：检测到人，等待2秒
    STATE_SHAKE  = 2   # 摇签：计数光敏脉冲
    STATE_REVEAL = 3   # 预言：展示结果

    def tick(self):
        if self.state == STATE_IDLE:
            if self.pir and self.light_delta > config.SENSE_THRESHOLD:
                self.enter_state(STATE_SENSE)
                return

        elif self.state == STATE_SENSE:
            if self.ticks_since_enter() > 2000:
                self.enter_state(STATE_SHAKE)

        elif self.state == STATE_SHAKE:
            self.count_light_pulses()
            if self.shake_count >= 3:
                self.generate_fortune()
                self.enter_state(STATE_REVEAL)
            elif self.ticks_since_enter() > 5000:
                self.enter_state(STATE_IDLE)  # 超时放弃

        elif self.state == STATE_REVEAL:
            if self.ticks_since_enter() > 10000 or not self.pir:
                self.enter_state(STATE_IDLE)
```

## 十、扩展想法

- 🎵 **加蜂鸣器**：预言揭晓时播放神秘音阶（幽灵探测器如果有蜂鸣器就能复用）
- 📜 **历史记录**：EEPROM 存最近 10 条预言，LCD 第五页翻看
- 🌐 **联网版**：MQTT 上传每日运势到 Web，"今日全站运势一览"
- 🌙 **夜间模式**：LCD 暗紫色主题，晚上更神秘
- 🃏 **塔罗牌版**：把一句话预言换成塔罗牌名 + 正逆位（22 张 × 2 = 44 种结果）
